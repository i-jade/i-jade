开始写作之旅

快速开始栏

你可以使用快速开始栏和 Dendron 进行交互。快速开始栏可以用来查找笔记和创建笔记。使用Dendron: Lookup(快捷键Ctrl+L/Cmd+L)打开快速开始栏。

创建笔记

在 Dendron 创建笔记的步骤如下：

  1. 使用快捷键Ctrl+L/Cmd+L打开快速开始栏；

  2. 在快速开始栏中输入笔记名（例如：recipes);

  3. 点击Create New;

  4. 按下Enter键，名字为 recipes.md 的笔记就创建好了。您可以在里面添加内容。

    🎗️ 小提示：使用快捷键Ctrl+Tab/Cmd+Tab来在编辑的笔记之间的快速跳转。
    📝 信息：笔记是 markdown 文件，请注意

    1. 笔记存储在本地
    2. 笔记可以跨操作系统，软件
    3. 让您的笔记快速导入导出，查看发布笔记到网站上
